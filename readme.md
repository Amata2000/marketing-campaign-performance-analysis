Marketing Campaign Performance Analysis
Project Overview
This project analyzes digital marketing campaign performance using real advertising data. The goal is to evaluate channel effectiveness, conversion efficiency, cost efficiency, and ROI, and to surface actionable insights that can guide marketing budget allocation and campaign optimization.
The analysis is implemented using Python, combining data cleaning, feature engineering, KPI calculation, and visualization into a modular and reproducible workflow.
Objectives
Measure campaign performance across key marketing metrics
Identify high-performing and underperforming segments
Evaluate cost efficiency and return on investment
Support data-driven marketing decisions
Dataset
Source: Marketing campaign performance data
Key Fields:
Campaign and ad identifiers
Demographics (age, gender)
Interests
Impressions, clicks, and spend
Total conversions and approved conversions
